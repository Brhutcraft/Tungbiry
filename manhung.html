Nút Bấm Hiệu Ứng Đẹp

Hướng dẫn: Sao chép và dán đoạn mã HTML vào thân trang của bạn, và CSS vào file stylesheet.

HTML


<button class="pretty-button">Click Me</button>

CSS


.pretty-button {
  background-color: #3498db;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 5px;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.pretty-button:hover {
  background-color: #2980b9;
}
